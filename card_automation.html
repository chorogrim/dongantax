핵심 기능

반복 작업을 한 번으로

매월 같은 가맹점, 같은 계정. 한 번 확정하면 다음번엔 자동으로 처리됩니다.

⚡
자동 파싱
우리카드·KB국민·하나·현대·IBK·롯데·삼성·신한·BC카드 9개 카드사 21종 포맷을 자동 감지해서 바로 전표로 변환합니다. 엑셀 여러 개를 한 번에 업로드해도 순서대로 자동 처리됩니다.
🧠
AI 자동 분개
가맹점명을 보고 복리후생비·차량유지비·접대비 등 계정과목을 자동 배정합니다. API 키 없이도 규칙 기반으로 동작합니다.
📚
가맹점별 룰 학습
확정 버튼 한 번으로 가맹점 → 계정 룰이 저장됩니다. 다음 달 같은 가맹점이 오면 즉시 자동 적용됩니다.
🔍
Opus AI 재검증
부가세 0원인데 과세유형이 맞지 않거나 불공제 대상인 경우 자동으로 플래그를 세우고 Opus로 재검증합니다.
🏷️
거래처 코드 관리
전표 확정 시 가맹점 코드를 자동 채번하고, 위하고 거래처 코드 파일을 가져와 일치·불일치·신규를 대조해 정리합니다.
☁️
Google Drive 자동 업로드
분개 룰을 버튼 한 번으로 Google Drive 지정 폴더에 자동 백업합니다.
📊
부가세 리포트
매입·매출 과세유형별 집계, 납부(환급)세액 자동 계산, 위하고 호환 엑셀 내보내기까지 한 번에.
🛡️
취소·공용카드 처리
취소 거래는 자동으로 🔒 잠금 처리되고, 공용카드 구분행은 해당 카드 전체에 공용 플래그를 적용합니다.
🛠️
카드사 규칙 직접 생성
지원하지 않는 카드사 명세서를 올리면 AI가 구조를 분석해 파싱 규칙을 JSON·MD로 만들어줍니다. 같은 카드사면 자동으로 기존 규칙에 병합됩니다.
사용 방법

5단계로 끝납니다

01
엑셀 업로드
카드사 명세서 엑셀 파일을 드래그하거나 버튼으로 올립니다. 여러 파일을 한 번에 선택해도 자동 처리됩니다.
02
AI 자동 분개
가맹점별 계정과목·과세유형이 자동으로 배정됩니다. 저장된 룰이 있으면 즉시 적용.
03
검토 및 확정
전표를 열어 계정을 수정하고 확정합니다. 확정 시 가맹점 룰과 거래처 코드가 자동 저장됩니다.
04
거래처 코드 정리
위하고에서 받은 거래처 코드 파일을 가져와 자동 대조하고, 신규 거래처만 골라 업로드 양식으로 내보냅니다.
05
내보내기
위하고 호환 엑셀, 분개 룰 JSON·MD, 부가세 리포트로 내보내거나 Google Drive에 자동 업로드합니다.
